Sakilol
======

Visualisation rules once post processed by AlainPeters lib


requirements
============

* compiler supporting C++11
* poco c++ lib
* AlainPeters lib


setup
=====

to make:
 ```$ make```

to make doc:
 ```$ make doc```

to clean:
 ```$ make clean```

to full clean:
 ```$ make raz```
